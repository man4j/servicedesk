Скрипты счетчика времени

Описание скрипта

Скрипт задает условия запуска/ приостановки/ возобновления/ остановки отсчета счетчика времени, см. .

Когда выполняется скрипт:

  • как только будет подобран счетчик времени для объекта;
  • каждый раз при изменении объекта.

Результат выполнения скрипта:

Если скрипт возвращает "true" (верно), то условие выполнено и производится запуск, приостановка, возобновление или остановка счетчика.

Если "false" (ложь), то счетчик остается в текущем состоянии. .

Переменные контекста и их значения

Глобальные переменные контекста:

  • user — пользователь, инициализировавший событие. Является объектом класса "Сотрудник" (employee)). Если событие инициализировал суперпользователь, то user=null.
  • ip — ip пользователя user.
  • api — содержит доступные вспомогательные методы. Например api.utils, api.ldap, api.timing,....
  • modules — содержит доступные модули, загруженные в интерфейсе технолога (набор библиотек написанных на groovy).
  • appVersion — версия приложения.
  • logger — объект консоли (для логирования), используется только при выполнении скриптов через консоль.
  • utils — Синоним api.utils.

 

  • subject — объект, атрибутом которого является счетчик;
  • oldSubject — старое значение объекта.

Примеры скрипта

  1. Скрипт условие начала отсчета счетчика времени:

    return (subject.state == 'registered' && oldSubject.state == 'resolved');

  2. Скрипт условие остановки счетчика времени:

    return (subject.state == 'closed');
